Custom style sheet

When output mode is html you can provide a custom style sheet code with which you can customize the formatted output.

With the custom css you can provide/override the style of all html elements.

When you provide a style for a specific format, it is appended to the global style.

Highlight use classes to highlight the source code:

body.hlCanvans
pre.hlDefault content
.hl.numNumbers
.hl.escEscape
.hl.strString
.hl.ppsPreprocessor
.hl.slcSingle line comment
.hl.comMulti line comment
.hl.ppcPostprocessor
.hl.optOperator
.hl.iplInterpolation
.hl.linLine numbers
.hl.kwaKewkord 1
.hl.kwbKewkord 2
.hl.kwcKewkord 3
.hl.kwdKewkord 4
.hl.kweKewkord 5
.hl.kwfKewkord 6

Customized styles are saved in ~/Library/Application Support/Syntax Highlight/Styles.
